# hyperdrive

# Objetivo do jogo:
O jogador controla uma nave que está sendo perseguida por naves policiais intergalácticas.
## O objetivo é:
Fugir pelo maior tempo possivel.
Desviar dos obstáculos.
Coletar recursos pelo mapa para desbloquear naves novas.
## Vence quem:
Sobreviver por mais tempo.

# Regras:
O jogador controla apenas uma nave
se a nave for atingida por tiros, perde vida 
se a nave colidir com os obstaculos ou as naves policiais perde vida e perde velocidade. 
quando zerar a vida, perde o jogo.
a polícia espacial fica mais rápida com o tempo.
## O jogador pode: 
Desviar de obstaculos.
atirar contra os policiais.
coletar itens no espaço.

# Personagens:
## Protagonista:
Um piloto fugitivo.
Um rebelde que gosta de bagunçar a ordem no espaço.
Nome é Max Rocket.
## Antagonistas:
Policia espacial.

# Artefatos:
## Intens de ajuda:
Carregardor para atirar.

# Fontes de receita:
Venda do jogo.
Compras dentro do jogo(novas naves).
